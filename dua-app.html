Introduction

Dua App is a production-grade Android Islamic companion built for Kheprx and shipped to real users. The core design principle was simple: every single feature must work without an internet connection. In practice that meant shipping a complete offline Quran database, a full Duaa and Azkar library, on-device prayer time calculations, and a GPS-powered mosque finder — all without a single network call for content.

Engineered on Clean Architecture with Jetpack Compose + Material 3, the app covers the full spectrum of daily Islamic practice: prayer scheduling with Azan notifications, Qibla direction via the device compass, Quran reading with transliteration and translation, categorised Duaa and Azkar collections, and an integrated Hijri calendar alongside Gregorian.

Contributions

  • Designed and implemented the full offline content system — Quran, Duaa, and Azkar databases stored locally in Room DB, eliminating any dependency on network availability.
  • Built on-device prayer time calculation using the device's GPS location and standard astronomical formulas, delivering precise Fajr–Isha schedules tailored to the user's exact coordinates.
  • Implemented a WorkManager-driven Azan notification system that schedules and fires the five daily prayer alerts reliably, even after device restarts.
  • Engineered a real-time compass-based Qibla pointer that reads the device orientation sensor and renders a live direction arrow toward Mecca with degree readout.
  • Integrated Google Maps SDK and FusedLocationProvider to surface the nearest mosques on an interactive map based on the user's live location.
  • Delivered a full Quran Reader with surah and ayah navigation, Arabic text, transliteration, and translation — all stored offline.
  • Built the complete Morning and Evening Azkar screens with Arabic text, transliteration, translation, and a counter to track repetitions.
  • Implemented a Hijri Calendar that displays the Islamic date alongside the Gregorian date throughout the app.
  • Applied modular project structure to keep features independently buildable and testable, reducing compile times and team friction.

Key Highlights

Offline-First

Room DB and DataStore power the entire app — Quran, Duaa, prayer calculations — with zero network dependency for any core feature.

Qibla Direction

Real-time compass-based pointer toward Mecca, updated live from the device orientation sensor with a precise degree readout.

Azan Notifications

WorkManager schedules all five daily prayer alerts on-device — reliable across reboots and background restrictions.

Mosque Finder

Google Maps SDK paired with FusedLocationProvider surfaces nearby mosques on an interactive map from the user's live GPS position.
