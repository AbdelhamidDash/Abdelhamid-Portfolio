FlutterColorPicker

FlutterColorPicker is an open-source Flutter package that provides a fully customizable, production-ready color picker widget. It supports HSV, RGB, and hex input modes, making it easy to drop into any app that needs color selection — from design tools to theming screens.

Available on GitHub
View on pub.dev

Introduction

Color picking is a deceptively complex UI problem — managing HSV gradients, hex validation, and real-time preview updates while keeping the widget composable and theme-aware. Most existing solutions were either too opinionated or too bare-bones to drop into a real product.

FlutterColorPicker is built as a pure Flutter widget with no native dependencies. It exposes a clean callback-based API, supports both light and dark themes out of the box, and is fully accessible with proper semantics labels.

Contributions

  • Built the HSV color wheel and saturation/value slider using Flutter's CustomPainter for pixel-accurate rendering.
  • Implemented bidirectional sync between HSV, RGB, and hex input modes so all three update in real time from any source.
  • Added alpha channel support with a checkerboard-pattern transparency preview.
  • Published to pub.dev with API documentation, screenshots, and a live example app.
  • Maintained compatibility across Flutter stable, beta, and master channels with CI checks on each release.
