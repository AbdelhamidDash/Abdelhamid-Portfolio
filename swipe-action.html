SwipeAction

SwipeAction is an open-source Android library that brings effortless swipe-to-action gestures to any RecyclerView. Developers can attach swipe callbacks, custom icons, and background colors to list items in just a few lines of code — no boilerplate, no hassle.

Available on GitHub

Introduction

SwipeAction was born out of frustration with the verbosity of Android's built-in ItemTouchHelper. Every RecyclerView swipe implementation required dozens of lines of boilerplate — custom drawing logic, icon positioning, threshold detection — repeated across every project.

This library abstracts all of that into a clean, chainable API. Attach it to any RecyclerView, define your swipe directions and actions, and the library handles rendering, haptic feedback, and callback delivery automatically.

Contributions

  • Designed and built the core swipe engine on top of ItemTouchHelper, handling multi-directional swipe detection and threshold logic.
  • Implemented custom Canvas drawing for icons and background color fills during the swipe gesture.
  • Added support for both left and right swipe actions on the same item with independent callbacks and styling.
  • Published to JitPack with a versioned release pipeline and a documented integration guide.
  • Wrote a sample app demonstrating all swipe configurations with a live preview.

Key Highlights

Zero Boilerplate

Attach swipe actions to a RecyclerView in 3 lines — no custom ItemTouchHelper subclass needed.

Custom Rendering

Full control over icon, background color, and corner radius via a simple builder API.

Bi-directional

Independent left and right swipe actions on the same item with separate callbacks.

Open Source

Published on JitPack with semantic versioning and a sample app for quick onboarding.
