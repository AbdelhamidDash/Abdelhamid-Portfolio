Introduction

IDH Chemist App is an Android POS application built for two of Egypt's largest medical testing lab chains — Alborg and Almokhtabar. Both labs previously maintained separate codebases for their customer-facing apps, leading to duplicated effort and inconsistent feature delivery.

The project consolidated both apps into a single, unified Android codebase using Android Build Flavors, allowing each brand to ship its own production app with distinct branding while sharing all core logic — authentication, caching, pricing, and POS integration.

Contributions

  • Consolidated two separate codebases into a single Android project using product flavors, eliminating duplicated logic and reducing future maintenance cost.
  • Stabilized the legacy apps with critical bug fixes to keep operations running while the unified codebase was in development.
  • Integrated with POS hardware for real-time price synchronization across lab service catalogs.
  • Built a secure authentication flow for lab staff with session persistence and access control.
  • Implemented offline caching with Room DB so core operations remain available without a network connection.
  • Developed a live location tracking feature allowing patients to follow their chemist in real time, with shortest-path routing from the chemist to the patient's location.

Key Highlights

Build Flavors
